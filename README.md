## Publish Maven Local

```bash
./gradlew fphi-selphi:publishToMavenLocal
./gradlew fphi-selphid:publishToMavenLocal
```

## Publish Maven Remove

```bash
./gradlew fphi-selphi:publish
./gradlew fphi-selphid:publish
```
